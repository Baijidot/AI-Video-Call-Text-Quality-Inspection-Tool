# AI 视频通话文本质检小工具

一个基于 Gradio 的轻量化 AI 文本质检工具，用于视频通话文本的质量检查。

## 功能特点

- ✅ 简洁的 Gradio Web 界面
- ✅ TXT 文件上传与解析
- ✅ 违规词/无效词自动检测
- ✅ 详细的质检报告生成
- ✅ 实时结果展示

## 项目结构

```
├── app.py                  # Gradio 前端界面（模块 1）
├── quality_check.py        # AI 质检核心逻辑（模块 2）
├── report_generator.py     # 简易报告生成（模块 3）
├── check_words.txt         # 违规/无效词库
├── sample_text.txt         # 示例测试文本
└── requirements.txt        # 依赖包
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置词库

编辑 `check_words.txt` 文件，格式如下：

```
违规词：敏感词1,敏感词2,禁止,违法；无效词：嗯,啊,呃,那个,这个,重复
```

### 3. 运行应用

```bash
python app.py
```

应用会自动在浏览器中打开，默认地址：http://127.0.0.1:7860

### 4. 使用说明

1. 点击"上传视频通话文本记录（TXT 格式）"上传文本文件
2. 查看上传状态确认文件已加载
3. 点击"开始质检"按钮
4. 在右侧查看详细的质检报告

## 测试

可以使用提供的 `sample_text.txt` 进行测试。

## 模块说明

### 模块 1：Gradio 界面（app.py）
- 顶部标题栏
- 左侧文本上传区
- 右侧结果展示区
- 底部质检按钮

### 模块 2：质检核心逻辑（quality_check.py）
- 词库加载与解析
- 文本分句处理
- 违规词/无效词匹配
- 异常处理机制

### 模块 3：报告生成（report_generator.py）
- 统计总句子数、合格数、问题数
- 计算质检通过率
- 问题类型分布统计
- 格式化报告输出

## 注意事项

- 确保上传的文本文件为 UTF-8 编码
- 词库文件 `check_words.txt` 必须存在
- 支持的文件格式：.txt
